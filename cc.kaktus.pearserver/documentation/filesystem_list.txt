

c -> channels
  - list of categories available
  - Channel information
    - c/Channels/info.xml
    - c/Channels/packages.xml - list of channel packages on the server
    - c/Channels/packagesinfo.xml - list all the packages with details and dependencies
  - Database-related packages
    - c/<channel name>/info.xml
    - c/<channel name>/packages.xml - list of database packages on the server
    - c/<channel name>/packagesinfo.xml - list all the packages with details and dependencies

m -> maintainers
  - m/allmaintainers.xml - short list of all maintainers
  - m/<maintainer name>/info.xml - info on a specific maintainer
p -> packages
  - p/packages.xml - list of the packages
  - p/<package name>/info.xml - information on the component
  - p/<package name>/maintainers.xml - list of maintainers
r -> releases
  - r/allreleases.xml - list all releases
  - r/<package name>/<version number>.xml - version details
  - r/<package name>/allreleases.xml - list all releases
  - r/<package name>/stable.txt - version number of the latest stable release
  - r/<package name>/latest.txt - version number of the latest release
